# Personal Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Animated sections
- Modern gradient hero section
- Project showcase grid
- Skills section
- Contact information

## Customization Guide

### 1. Update Personal Information

Edit `index.html` and replace:
- "Your Name" with your actual name
- "Web Developer | Designer | Creator" with your title/role
- Update the about section with your bio
- Add your email, LinkedIn, and GitHub links in the contact section

### 2. Add Your Projects

In the projects section, update:
- Project titles
- Project descriptions
- Add links to live demos and GitHub repos
- Replace placeholder images with actual project screenshots

### 3. Update Skills

Modify the skills section to reflect your actual skills and technologies.

### 4. Customize Colors

In `style.css`, update the CSS variables:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* Change these to your preferred colors */
}
```

## Deployment to Netlify

1. Go to [Netlify](https://www.netlify.com/)
2. Sign up or log in
3. Click "Add new site" → "Import an existing project"
4. Connect to GitHub and select this repository
5. Deploy settings:
   - Build command: (leave empty)
   - Publish directory: (leave empty or use `/`)
6. Click "Deploy site"

Your site will be live in minutes!

## Local Development

Simply open `index.html` in your browser to view the site locally.
